### UNIX System Directories

* `/bin` : Unix binaries
* `/sbin` : System binaries
* `/usr` : The User directory (Third-party)
* `/usr/lib` : Shared objects (Third-party)
* `/etc` : Et Cetera. It has most of the system configuration files.
* `/dev` : BSD device files. It represents hardware devices on the system. 
* `/tmp` : Temporary directory
* `/var` : Various.
